# Fun With Particle IO API

A web portal for Particle.io users to manage their IoT devices.

###Getting Started###

	npm install

###To Start Dev Server###

    npm start

Go to:

    http://localhost:8080

###Tech Stack###

    React
    React Router
    ES6
    Webpack
    Webpack-dev-server
    Axios
    Bootstrap
    Mocha
    Chai
